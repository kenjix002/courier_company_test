# Courier Company App

## About

This application consists of 3 parts:

- client (React)
- server (NodeJS)
- database (MySQL)

## Run

To start the application, run the following in the root of the application folder

```
docker compose up -d
```

### Login credential

To start, do use the following users. One as a admin, while the other as a driver

```
User:   Admin
Pass:   Admin@123


User:   Driver
Pass:   Admin@123
```
